# Plan van Aanpak Iteraties
Ik heb meerdere malen feedback gehad op mijn plan van aanpak, aangezien we wekelijks moesten presenteren hoever we waren. Veel feedback commentaar is minimaal en zal ik voor dit stukje buiten beschouwing laten. Wel bespreek ik de grotere feedback momenten.

### Iteratie 1
In mijn eerste iteratie van het pva heb ik gewoon een pva gemaakt. Bij het maken van de eerste versie van het pva heb ik een paar bronnen gebruikt:
- In de expert workshop van Lamia hebben we een format gekregen voor het pva. Ik heb dit format gebruikt als basis voor mijn pva.
- De beoordelingscriteria in DLO geven een algemeen idee van wat er precies verwacht wordt. Deze criteria ik heb ook gebruikt om mijn pva vorm te geven.
- Ook heb ik feedback van eerdere pva's gebruikt om dit pva optestellen. aangezien ik voor beide voorgaande pva's onvoldoende had gescoord had ik erg veel feedback die ik kon gebruiken.

Met al deze gegeven info bleef er weinig ruimte voor aannames. Hierdoor werd het pva een kwestie van invullen wat er van je gevraagd wordt. Ik heb letterlijk dat gedaan, niks meer, niks minder en heb daarom logischerwijs voor elke leeruitkomst "Op Niveau" gescoord.
